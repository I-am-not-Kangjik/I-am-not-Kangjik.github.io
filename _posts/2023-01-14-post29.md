## jekyll bundler 설치하기

### 1. brew 최신으로 업데이트

```bash
> brew update
```

### 2. ruby-build 설치

```bash
> brew install rbenv ruby-build
```

### 3. rbenv 설치 확인

```bash

> rbenv versions
* system
```

### 4. ruby 설치

```bash

> brew install ruby
```

### 8. vs code로 경로설정

```bash
> code ~/.zshrc
```

### 9. /.zshrc 에 아래 내용 복붙

```
[[ -d ~/.rbenv  ]] && \
  export PATH=${HOME}/.rbenv/bin:${PATH} && \
  eval "$(rbenv init -)"
```

### 10. 경로 설정 적용하기

```bash
> source ~/.zshrc
```
